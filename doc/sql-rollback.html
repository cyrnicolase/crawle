ROLLBACK

名称

ROLLBACK -- 中止当前事务

大纲

ROLLBACK [ WORK | TRANSACTION ]

描述

ROLLBACK回滚当前事务并且导致 该事务所作的所有更新都被抛弃。

参数

WORK
TRANSACTION

可选关键词，没有效果。

注解

使用COMMIT可成功地终止一个事务。

在一个事务块之外发出ROLLBACK会发出一个 警告并且不会有效果。

示例

要中止所有更改： 

ROLLBACK;

兼容性

SQL 标准只指定了 ROLLBACK和ROLLBACK WORK两种形式。除此之外，这个命令完全符合 标准。
