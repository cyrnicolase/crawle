SPI_exec

名称

SPI_exec -- 执行一个读/写命令

大纲

int SPI_exec(const char * command, long count)

描述

SPI_exec和 SPI_execute相同，但后者的 read_only参数的值总是取 false。

参数

const char * command

包含要执行的命令的字符串

long count

要返回的最大行数，0表示没有限制

返回值

见SPI_execute。
