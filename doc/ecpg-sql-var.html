VAR

名称

VAR -- 定义一个变量

大纲

VAR varname IS ctype

描述

VAR命令分配一个新的 C 数据类型给一个主变量。主变量必须之前在一个声明节中声明过。

参数

varname

一个 C 变量名。

ctype

一个 C 类型说明。

例子

Exec sql begin declare section;
short a;
exec sql end declare section;
EXEC SQL VAR a IS int;

兼容性

VAR命令是一个 PostgreSQL 扩展。
