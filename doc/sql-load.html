LOAD

名称

LOAD -- 载入一个共享库文件

大纲

LOAD 'filename'

描述

这个命令把一个共享库文件载入到PostgreSQL服务器 的地址空间中。如果该文件已经被载入，这个命令什么也不会做。只要调用 包含 C 函数的共享库文件中的一个函数，这些共享库文件就会被自动载入。 因此，一次显式的LOAD通常只在载入一个通过 "钩子"修改服务器行为而不是提供一组函数的库时需要。

文件名以CREATE FUNCTION中的共享库名称相同的方式 指定。特别地，我们可以依靠搜索路径和对系统标准共享库文件名扩展的自动 补全。有关于此的更多信息可见第 36.9 节。

非超级用户只能把LOAD应用在位于 $libdir/plugins/中的库文件 — 指定的 filename 必须正好以该字符串开始（确保在那里只安装了 "安全的"库是数据库管理员的责任）。

兼容性

LOAD是一种 PostgreSQL扩展。
