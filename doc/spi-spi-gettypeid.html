SPI_gettypeid

名称

SPI_gettypeid -- 返回指定列的数据类型的OID

大纲

Oid SPI_gettypeid(TupleDesc rowdesc, int colnumber)

描述

SPI_gettypeid返回该指定列的数据类型的 OID。

参数

TupleDesc rowdesc

输入行描述

int colnumber

列号（从 1 开始计）

返回值

指定列的数据类型的OID，或者出错时返回 InvalidOid。出错时， SPI_result会被设置成 SPI_ERROR_NOATTRIBUTE。
